# Design Doc 索引

设计文档（Design Document）— 记录模式的实现细节、设计权衡、已知局限。

- **格式与变更门槛**：见 [DOCUMENTATION.md](../DOCUMENTATION.md#5-design-doc--设计文档)
- **模板**：[_template.md](_template.md)
- **与 Spec 的关系**：[Spec](../spec/) 描述 **what**（契约），Design Doc 描述 **how** + **why**（实现）
